Como usar Projetos?

1. Criação: Crie um projeto para cada destino específico de recursos (ex: MENIBRAC, Anistia 30%, Reforma do Telhado).

2. Lançamentos: Ao cadastrar lançamentos com categoria "Outras Ofertas" ou "DESTINAÇÃO", selecione o projeto correspondente.

3. Relatórios: Acesse "Caixa de Destinações" para ver o saldo de cada projeto separadamente.

4. Transparência: Todos os lançamentos aparecem no Caixa Geral, mas são controlados por projeto no Caixa de Destinações.
